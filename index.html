Personal Projects

Memory Game

In this interactive Memory Game, players are challenged to remember and select the correct sequence of numbers. Five numbers, ranging from 1 to 5, briefly appear on the screen for 3 seconds. After they disappear, players must select the boxes in the correct order from 1 to 5 to win. If successful, they win the game; otherwise, the game resets and they can try again. This project highlights my skills in JavaFX, object-oriented programming, and UI design, offering a simple yet engaging test of memory and cognitive skills.
